How to View Your Report

  1. Navigate to the "Actions" tab in your GitHub repository.
  2. Select the "Security Audit" workflow from the list on the left.
  3. Click the "Run workflow" dropdown button. You will be prompted to enter the domain to scan.
  4. Once the workflow is complete, a "ReporteDeSeguridad" artifact will be available for download on the workflow summary page. This file contains your HTML report.
